# Brand assets

- `logo.webp` / `logo.png` — primary circular emblem with transparent background (nav, hero)
- `logo-256.webp` — compact transparent emblem for dense UI
- `social-share.jpg` — default Open Graph / Twitter / social link preview image
- Favicons live in `app/icon.png` and `app/apple-icon.png` (Next.js metadata routes)
- Regenerate transparency after replacing the source art: `python scripts/make-logo-transparent.py`
